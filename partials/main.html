Model Driven: Less Code, More Features

Focus on your application design, and let erocci deals with the runtime. In most of cases, you don't need to code at all.

Self-Described

REST API designed with erocci are self-described: writing clients for your application has never been so easy.

Multi Protocol

In addition to HTTP protocol family, erocci can expose your API on XMPP or D-Bus. You can also extend to your own protocol through a simple API.

Scalable

Thanks to erlang/OTP platform, erocci can scale from embedded devices to a full distributed cluster.

Standard

erocci is fully compliant with the OCCI standard, which means you don't have to care with implementation details, but just on your application.

Open Source

erocci is released under Apache Software License, version 2.0.
